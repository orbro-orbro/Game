## 1. 跳跃

### 1.1 基础参数

| 参数 | 你的选择 | 备注 |
|------|----------|------|
| 跳跃高度（像素/tile） | 【4个主角高度】 | |
| 跳跃到达顶点时间（帧） | 【16】 | 60fps 基准，16帧≈0.27秒到顶 |
| 空中水平移动速度 | 【地面速度 × 90%】 | 空中比地面略慢 |
| 空中水平加速度 | 【地面加速度 × 70%】 | 空中调整比地面慢，有惯性感 |
| 落地惯性保留比例 | 【0.3】 | 0~1，保留少量惯性，手感流畅但不滑 |

### 1.2 跳跃缓冲 (Coyote Time)

角色离开平台边缘后，多少帧内仍可按下跳跃？

> 参考：Celeste = 5帧，SMB = 10帧。太长会飘，太短会感觉"吞输入"。

【5】帧
> 与 Jump Buffer 同为 5 帧。Coyote Time + Jump Buffer 双 5 帧 = 10 帧宽恕窗口，跟手但无漂浮感。

### 1.3 跳跃高度分段

| 模式 | 行为 |
|------|------|
| 短按跳 | 到达全高的【 70】% 后开始下落 |
| 长按跳 | 到达全高 |

是否启用短按/长按区分？ 【是 】

> 如果选"否"，则每次跳跃都到全高，操控更直接但失去微调空间。

### 1.4 跳跃输入缓冲 (Jump Buffer)

在落地前多少帧按下跳跃，落地后自动起跳？

> 参考：Celeste = 5帧。配合 Coyote Time 一起决定"手感是否跟手"。

【 5】帧

## 2. 冲刺 (Dash)

### 2.1 方向

冲刺方向选择：

- 【 】八向（上下左右+四斜角）
- 【 是】四向（仅上下左右）
- 【 】四向但不可上冲
- 【 】仅水平

### 2.2 距离与时间

| 参数 | 你的选择 |
|------|----------|
| 冲刺距离（像素/tile） | 【4个主角宽度 】 |
| 冲刺持续时间（帧） | 【8】 | 8帧覆盖4个主角宽度，先快后慢 |
| 冲刺速度曲线 | 【先快后慢】 |
| 冲刺后摇时间（帧） | 【4】 | 冲刺结束后4帧不可操作，避免连冲 |

### 2.3 次数与恢复

每段跳跃中可冲刺次数：**【 1】次**

恢复条件（可多选）：
- 【 是】落地时恢复
- 【 】攀墙时恢复
- 【 是】触碰特定物体恢复（如绿色水晶）
- 【 】穿过检查点恢复

> Celeste 是 1 次 + 落地/攀墙/水晶恢复。次数越多，关卡容错越高但深度越低。

### 2.4 无敌帧

冲刺过程中是否无敌？
- 【 】全程无敌
- 【 】前半帧无敌
- 【是 】无无敌帧
- 【 】仅对特定伤害类型无敌

- 【 是】触碰时需按住方向键朝向墙

### 3.2 耐力系统

| 参数 | 你的选择 |
|------|----------|
| 攀墙最大时间（秒） | 【4秒 】 |
| 是否显示耐力条 | 【是 】 |
| 耐力耗尽后行为 | 【缓慢下滑】 |

### 3.3 攀墙跳跃

抓住墙壁后，按下跳跃键的行为：

- 【是 】固定角度弹出（约 45°）
- 【 】根据摇杆方向决定弹出角度
- 【 】水平弹开（与墙垂直）

| 参数 | 你的选择 |
|------|----------|
| 弹墙跳跃高度 | 【 3个主角高度】 |
| 弹墙跳跃水平距离 | 【三个主角宽度 】 |
| 弹墙后是否消耗冲刺 | 【 否】 |
| 弹墙后能否再次抓同一面墙 | 【否】 | 需先落地或冲刺离开后才可再抓，防止无限垂直攀爬 |

### 3.4 滑墙

下滑墙面时的速度：【2】像素/帧
> 缓慢下滑，给玩家反应时间。60fps 下约 120px/秒。

是否可以控制下滑速度？ 【否】

## 5. 关卡结构设计

### 5.1 关卡组织形式

- 【 】章节制（Chapter A/B/C 面）
- 【 是】纯线性关卡链
- 【 】世界地图选关

### 5.2 关卡内结构

每个关卡的目标规模：

| 参数 | 你的选择 |
|------|----------|
| 普通关卡房间数 | 【15】 |
| Boss 关房间数 | 【无】 |
| 单个房间平均通关时间（秒） | 【9】 | 15间×9秒≈2分15秒/关 |
| 单关总时长目标（分钟） | 【2~2.5】 | 首通约2~2.5分钟，熟练后可压缩 |

### 5.3 重试机制

死亡后：
- 【是】立刻从本房间起点重生
- 【 】从本关起点重生
- 【 】从最近检查点重生

重生等待时间：【800】毫秒
> 改为房间级重生 + Celeste 节奏。800ms 既不打断心流，也避免了全关重开的挫败。

> Celeste 是房间起点重生，等待 ~800ms。越快越好，但需要给玩家一个"呼吸"的间隙。
