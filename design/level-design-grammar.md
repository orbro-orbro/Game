## 2. 机制教学顺序（首发关卡设计）

> 原则：一次只教一个机制，每个机制用 3 个房间练熟，再加第二个。红点放在必经之路上引导方向。

```
关卡 1：入门 — 15 间

  房间 1~2：移动+跳跃基础
  ├─ 1：平地跑+第一个坑。红点悬在对面引导第一次跳跃，学会按下跳跃键。
  └─ 2：不同高度平台+长按/短按区分。低平台用短按，高平台用长按。

  房间 3~5：引入冲刺
  ├─ 3：水平冲刺。前方有水晶+红点，提示"冲刺穿过去"。
  ├─ 4：空中冲刺。高台在地面跳不到的位置，必须跳起后空中冲刺上去。
  └─ 5：方向冲刺。头顶有尖刺+脚边有尖刺，用上下冲刺避开。

  房间 6~8：引入攀墙
  ├─ 6：垂直死路。前方无路，红点悬在高处——必须跳向墙壁、抓住、弹跳。
  ├─ 7：双墙交替。左右墙来回弹跳上升。此处自然教会"不可再抓同面墙"。
  └─ 8：攀墙+冲刺。抓住墙后冲刺到水晶，再抓对面墙继续上升。

  房间 9~10：引入动态元素
  ├─ 9：移动平台。站上去→观察节奏→跳到下一个。
  └─ 10：碎落平台。站上即碎，必须快速离开。

  房间 11~13：机制组合
  ├─ 11：冲刺+攀墙。空中冲刺接墙，弹跳后无缝再冲。
  ├─ 12：移动平台+攀墙。在移动平台上跳向墙壁。
  └─ 13：碎落平台+冲刺。碎落开始→冲刺脱离。

  房间 14~15：综合测试
  ├─ 14：长序列——无新教学，要求跳+冲+攀墙+动态平台连续执行。
  └─ 15：终点在望——所有机制全用上的短小高潮，过关。
```

## 3. 难度曲线设计

### 3.1 全局难度维度

并非所有"难"都一样。明确每个维度在关卡中的权重：

| 难度维度 | 阶段 1（关 1~3） | 阶段 2（关 4~6） | 阶段 3（关 7~N） |
|----------|-----------------|-----------------|-----------------|
| 操作精度要求 | 低（平台≥3格宽） | 中（≥2格宽） | 高（1格宽/像素级） |
| 反应时间窗口 | 宽松（∞） | 中（移动平台） | 紧（连冲+移动障碍） |
| 机制组合数 | 1~2 | 2~3 | 3 全部 |
| 序列长度 | 短（一屏内 2~3 动） | 中（一屏内 5~7 动） | 长（需跨屏连段） |
| 检查点惩罚 | 0（一屏） | 0 | 0（始终房间级） |

### 3.2 关卡数量目标

游戏总关卡数：【8 】

> 建议：首发版本 8 关（含 1 教学关 + 7 正式关）= 约 2.5 小时内容。可接受？

## 5. 红点放置原则

> 红点 = 纯成就收集品，不锁内容。

| 原则 | 说明 |
|------|------|
| 必经之路上 | 红点放在通关路径上，充当"路标"引导玩家方向 |
| 直觉可及 | 玩家看到红点就知道自己在正确路线上 |
| 每间 0~3 个 | 教学房可密集（2~3个/间），后期可稀疏（0~1个/间） |
| 死亡丢失本间 | 死亡后丢失该房间内收集的红点，鼓励无伤通关 |
| 全收集成就 | 集齐一关所有红点 = 该关全收集标记 |

> 与 Celeste 草莓的核心区别：红点不在偏离路线上，而是路径本身的一部分。全收集的挑战来自"不死"，而非"找到"。

## 6. 关卡主题规划

既然世界观"明亮+极简"，每大关可以有一个视觉主题，仅作区分：

| 关 | 主题 | 新机制/元素 | 设计重点 |
|----|------|-------------|----------|
| 1 | 草原/平原 | 无（纯教学） | 机制教学，红点引导，容错极高 |
| 2 | 森林 | 碎落平台 | 节奏感——"站上就要动" |
| 3 | 地下洞穴 | 移动平台 | 时机感——"观察再行动" |
| 4 | 云端/日落 | 冰面 | 惯性控制——"松手≠停" |
| 5 | 雪山 | 尖刺密度↑ | 精确性——平台变窄，尖刺变密 |
| 6 | 庙宇 | 绿色水晶连冲 | 连冲节奏——"一冲接一冲" |
| 7 | 城市/屋顶 | 无新元素 | 综合精度——最短窗口，最高密度 |
| 8 | 虚空/星空 | 无新元素 | 最终考验——全机制组合，最长序列 |

> 主题只需调色板+背景图切换。新元素分布在前期关卡中，后期关卡不再加新东西，而是在"排列组合"上做深度。

> 即使无新机制，主题切换也能提供视觉新鲜感。每关只需调色板+背景图不同即可。

## 7. 设计自检问题

每设计一关后检查：

1. 这关引入了什么新概念？前面的关卡教过吗？
2. 玩家在房间 N 失败后，重试路径是否公平？（手感，非视觉）
3. 有没有"假难度"——因为看不清、操作延迟、摄像机问题导致死亡？
4. 红点是否在玩家视线可及但路径稍远处？
5. 连玩 5 间后的操作疲劳度？需要"呼吸房"吗？
